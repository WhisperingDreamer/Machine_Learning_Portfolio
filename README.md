## 📂 Проекты  

### 1. Прогноз выбора вида транспорта ([репозиторий](https://github.com/WhisperingDreamer/ML-course-project))

**Суть:** Сравнение эффективности ручного и автоматического машинного обучения для оптимизации логистики

**Задача**: Построить модель прогнозирования выбора вида транспорта и сравнить эффективность ручной настройки моделей с AutoML.

**Методы**:
- EDA (Pandas, NumPy)
- Feature Engineering (создание временных и сезонных признаков)
- Модели (CatBoost, Случайный лес, Логистическая регрессия)
- Различные инструменты (Кросс валидация, подбор гиперпараметров (Optuna), регуляризация весов, стэкинг)

**Результат**:
- Лучшая модель: **Стэкинг**(все три модели) (f1: 0.880948, logloss: 0.396613)
- **AutoML** стал лучше ручных моделей (logloss: 0.110294)
- Ключевые факторы: количство поездок в день, дистанция поездки, время поездки, личное отношеие к мопеду, скутеру


### 2. Тематическое моделирование новостей водного транспорта (BERTopic, LDA) ([репозиторий](https://github.com/WhisperingDreamer/NLP-course-project))

**Суть:** Сравнение классических и нейросетевых методов тематического моделирования для автоматической рубрикации отраслевых новостей (водный транспорт)

**Задача:** Проанализировать поток сообщений Telegram-канала о водном транспорте и выделить ключевые темы для мониторинга отраслевой повестки

**Что сделано:**
- Сбор и парсинг данных через Telethon (парсинг сообщений, удаление ботов и дублей)
- Очистка и нормализация текстов (лемматизация через **pymorphy3**, удаление стоп-слов, шумовых токенов и эмодзи через demoji)
- Обучение **LDA-модели** с подбором числа тем по когерентности, визуализация через pyLDAvis
- Применение **BERTopic** для получения интерпретируемых тем поверх эмбеддингов sentence-transformers
- Анализ динамики тем по временным меткам сообщений

**Результат:** построен интерактивный дашборд распределения тем, выявлены 6 устойчивых тематических кластеров (нормативная база, происшествия, инфраструктура, экология и др.). Модель может использоваться для автоматической классификации новостей в отраслевых медиа
